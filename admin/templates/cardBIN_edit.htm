卡BIN
卡类型
商城售价策略 1.19公式 0.97公式    两种公式都是由添加供应商的设置的的，商城比例(1.19)、商城比例(0.97)
销售卡类型（次卡/点卡） 点卡 次卡    次卡是以次为单位销售，只应用与在线选座
警戒线上限(次卡)    商城售价在警戒线上限之上，超出的点数按1:1比例，支付宝补差价，（扣点 = 警戒线上限）只有次卡才生效
警戒线下限(次卡)    商城售价在警戒线上限之下，直接扣除卡点，无需支付宝补差价操作，（扣点 = 警戒线下限）只有次卡才生效
警戒线(点卡)    卡实际每点售价小于警戒线时，售价 = （规格价 x 售比 X 卡比 x 单品比 x 每点应售价）/ 每点时间售价 只有点卡才生效
